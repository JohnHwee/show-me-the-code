**第 0005 题：**二分查找

    int binSearch(T _elem, T e, int lo, int hi)

语义约定：

在有序区间 V[lo, hi) 中，确定不大于 e 的最后一个元素。

要求：

尽可能让 O(clogN) 的系数 c 尽可能的小。
